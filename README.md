# Daily Gross Analytics Dashboard

A modern, responsive dashboard for tracking daily teller performance across different units.

## Features
- **Real-time Data**: Fetches data directly from Google Sheets CSV.
- **Visual Analytics**: Interactive performance trend chart using Chart.js.
- **Unit Breakdown**: Automated grouping and summarization by unit/supervisor.
- **Detailed View**: Full searchable table of all transactions.
- **Modern UI**: Dark-themed, premium design with responsive layout.

## How to Run
1. Open `index.html` in any modern web browser.
2. The dashboard will automatically fetch and process the latest data.
3. Use the sidebar to switch between **Overview** and **Details**.
4. Use the search bar in the **Details** tab to find specific tellers.

## Technologies
- HTML5 / Vanilla CSS
- Vanilla JavaScript
- [Chart.js](https://www.chartjs.org/) for data visualization
- [PapaParse](https://www.papaparse.com/) for CSV processing
